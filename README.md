## 如何更新本手册--？

1. 修改要更新的章节对应的 md 文件
2. 执行根目录下的 bash 文件：`./merge-man.sh`，将最新的各章节内容合并到`mnos-user-manual.md`中
3. 将各个 md 文件修改更新到 git 仓库
